🩺 Multiple Disease Prediction System

A Streamlit-based Machine Learning web app that predicts diseases like diabetes and heart disease based on user input medical data.

🚀 Features
Predicts multiple diseases using ML models
Simple Streamlit UI
Fast and interactive results
🛠️ Tech Stack

Python, Streamlit, Scikit-learn, Pandas, NumPy

📦 Installation
pip install -r requirements.txt
▶️ Run App
streamlit run "multiple disease pred.py"
📁 Project
Multiple ML models trained on medical datasets
User-friendly prediction interface
